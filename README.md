# 🚧 AI Powered Pothole Detection System

An intelligent road monitoring system that automatically detects potholes using **YOLO Deep Learning** and provides real-time detection through images, videos, and live webcam streams.

This system helps improve **road safety and smart city infrastructure** by identifying road damages automatically.

---

## 🚀 Features

• AI based pothole detection using YOLO  
• Image upload detection  
• Video pothole detection  
• Live webcam detection  
• Real time detection visualization  
• Heatmap visualization for pothole locations  
• Interactive dashboard UI  
• Flask backend with HTML/CSS/JS frontend  

---

## 🧠 AI Model

This project uses **YOLO Object Detection Model** trained on a pothole dataset.

Model detects potholes with bounding boxes and confidence scores.

Framework used:
- Ultralytics YOLO
- PyTorch

## ⚙️ Installation

Clone the repository

```
git clone https://github.com/yourusername/AI-Pothole-Detection-System.git
```

Navigate to project folder

```
cd AI-Pothole-Detection-System
```

Install dependencies

```
pip install -r requirements.txt
```

Run the application

```
python app.py
```

Open browser

```
http://127.0.0.1:5000
```
